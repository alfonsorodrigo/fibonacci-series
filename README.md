# fibonacci-series

> django application to generate the fibonacci series

## Quick Start Backend

```bash

cd backend/src
Required Python 3+

# Install dependencies for server
pip install -r requirements.txt

# Run the server
python manage.py runserver

# http://localhost:8000
```

## Quick Start Frontend

```bash

cd frontend/gui

# Install dependencies for server
npm install

# Run the server
npm run start

# http://localhost:3000
```
